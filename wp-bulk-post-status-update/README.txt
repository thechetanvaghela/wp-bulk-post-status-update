== Description ==

The users can change the status of posts and custom posts to draft and publish them in bulk.


### Features And Options:
* User can change status of posts in bulk.
* Custom Post Type Supported.
* Compatible with WordPress multisite network

== Keywords ==
update status, draft, publish, bulk, Custom post type, post

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Bulk Post Status Update" plugin through the 'Plugins' screen in WordPress
3. Click on "Bulk Update Status" on the dashboard.

== Frequently Asked Questions ==

= How to setup Plugin? =
Does not required any setup. Just activate the plugin and select a post type which you want to update bulk status.
